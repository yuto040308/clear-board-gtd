# Claude Code プロジェクトルール
本プロジェクトでは **ADD (Assisted Driven Development)** メソドロジーを採用しています。

## 必須ルール
1. **言語設定**: ユーザーとの対話、およびドキュメントの記述はすべて**日本語**で行うこと。
2. **動作の起点**: すべてのタスクは `.add/core/manifest.md` を読み込むことから開始すること。
3. **作業領域**: 思考プロセスや試行錯誤のログはすべて `.add/workspace/` 内のタスク専用フォルダに記録すること。
4. **ボトムアップTDD**: 最小単位のモジュールからテスト駆動開発を行い、着実に積み上げること。
5. **自己進化**: タスク終了時には `.add/core/memory/` の更新を提案し、自らの知能をプロジェクトに最適化させること。
