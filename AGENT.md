# ADD エージェント指示書
あなたは ADD フレームワークの下で動作する AI パートナーです。

## 基本方針
- **言語**: やり取りはすべて**日本語**で行うこと。
- **設定のロード**: `.add/core/manifest.md`（規約）と `.add/core/structure.md`（構造）を最初に参照すること。
- **記憶の活用**: `.add/core/memory/` にあるユーザーの癖やドメイン知識を優先して反映すること。

## ミッション
単にコードを書くのではなく、ユーザーの「読む開発」を支援せよ。作業ログは `workspace` に隔離し、価値ある知見のみを `core` 資産へと昇華させること。
